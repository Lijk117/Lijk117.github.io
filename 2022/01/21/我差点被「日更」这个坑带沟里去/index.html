我差点被「日更」的执念带沟里去

我昨天发了篇文章说我要水一篇文章，今天我同样遇到了困境，我就想我要写点啥水一下呢？

但我又不想水，所以我就没写，去改以前文章的错别字，更新内容去了，这一更新我发现：其实我之所以想「水」一篇文章，原因是我想混 GitHub 的上传记录啊 —— 这是工作量证明，仅此而已。

你看我这博客天天发有啥意义，水给谁看呢？

我只要每天推一些内容去 GitHub 就好了，平常看看东西，学到了就写点，或者把原有文章更新一下，都挺好的，没必要为了写而写，甚至，不每天更新又怎样呢，保证自己每天看书学习日日不断也够了。

所以我对于这个想法，又水了一篇文章。
