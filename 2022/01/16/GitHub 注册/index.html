GitHub 注册

GitHub 相当于程序员的小红书，但程序员也可以注册小红书，你不是程序员，不妨碍你注册一下 GitHub，毕竟你虽然不是程序员，你要做的事情也确实是程序员常做的。

点击下面👇链接注册 GitHub：

点我注册

其实不难，但因为是全英文，所以我也走一遍流程：

因为是国外网站，所以加载起来可能会有点慢，耐心等待，有条件的话可以使用代理来加速，或者去看我的另一篇文章 加速国内GitHub访问.md

点击Sign up

输入邮箱后点击 continue

如果因为网络问题点不动，就耐心等待一下，我测了测确实是要等一些时间，可能有五分钟，每一步都是

输入密码和 username，耐心等待 Continue

等了一会，我通过点击 [Enter your email] 、[Create a password] 下面的内容切换，算是触发了

继续按钮

这三个都完成，下面的是要不要接收信息，输入 n 就好，拒绝，没必要收广告。

点击创建

输入邮箱码

这就注册完了，后面就是对你的调研

随便填，然后选择免费的版本，free 就可以了。
