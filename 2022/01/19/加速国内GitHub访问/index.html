加速国内 GitHub/GitBook 访问

由于一些特别原因，很多国外网站中国用户访问不了，或者访问非常慢，但这东西又是必须要用的，咋办？

可以这么做。

针对 mac 用户：

打开你的访达 ，同时按shift+command+G , 输入：

/private

之后，按 Enter, 进入 Private 文件夹

文件夹长下面这个样子，在 etc这个文件这里，点击右键，有个显示简介

你要按照下图的顺序，先把小锁头🔒解锁🔓，然后给 everyone 这个用户「读和写」权限，解锁需要密码，然后再把这个锁头给锁上。

这一步完成后，就双击进入到 etc 这个文件夹里，找到一个叫hosts的文件,也是右键点击显示简介，一样操作，也要给「读与写」权限，保存

给了 hosts 文件读与写的权限之后，你把它打开，如果还是现实已锁定的话，你就点一下这个小箭头，把「锁定」这里给他解锁。

然后就可以编辑了，把我下面列的内容复制到这个文件的最下面，点击保存（其实输入后直接关掉就是保存）

140.82.114.4 github.com

185.199.108.153 assets-cdn.github.com

185.199.109.153 assets-cdn.github.com

185.199.110.153 assets-cdn.github.com

185.199.111.153 assets-cdn.github.com

199.232.69.194 github.global.ssl.fastly.net

75.2.70.75 gitbook.com

99.83.190.102 gitbook.com

这些都操作完，你可以再用 Safari 浏览器尝试登陆 GitHub 了，应该会快一些：

www.github.com

当然，一本万利的办法还是解决掉网络环境问题。
